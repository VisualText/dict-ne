# dict-ne
Nepali Dictionary

This folder contains analyzer that is built to parse Nepali wikitext, develop Knowledge base
and write a record structure in XML format

## Procedure to Add Nepali Word to Wiktionary ##
```
Please visit the page: http://nepalinlp.org/procedure-of-word-entry-to-nepali-wiktionary/ to get familiar with the procedure to add word and its content to the Nepali Wiktionary in detail.
```

```

==पदवर्ग==

===अर्थ १===
# [[प्रशस्त]] 
# [[चाहिएजति]] 
# [[पुग्दो]]

====उदाहरण==== 
# गरीब देशहरुमा अधिकांश बच्चाहरुले पर्याप्त मात्रामा पोषित खाना खान पाउदैनन।
# एउटा मान्छेलाई दिनको ५-६ लिटर पानी खानु पर्याप्त हुन्छ।

===अर्थ २===
# [[पाइएको]] 
# [[प्राप्त]]

====उदाहरण==== 
# रामले छोरा पर्याप्तगर्नकालागि हवन गराएका थिए।

# [[समर्थ]]

====उदाहरण==== 
# मनु पढ्नमा पर्याप्त भएकोलेगर्दा अमेरिकामा पूर्ण छात्रवृतिमा उच्च शिक्षा पढ्न पाई। 

==समानार्थी शब्द== 
[[काफ़ी]], [[लात्तिकुत्ती]]

```
